Source code of Karel robot. 

# How to run
- Restore packages
  - `npm install`
- Build
  - `npm run devBuild`
- To run and open in browser
  - `npm run start`

# Deploy to production
- Create a release in GitHub
- TravisCI will do the deploy to production GitHub pages repo
